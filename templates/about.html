Acerca de Code Sender

¿Qué es Code Sender?

Code Sender es una herramienta web diseñada para fortalecer la integridad y el anonimato en cualquier proceso de recolección de datos, como encuestas, votaciones o formularios de registro.

¿Qué problema soluciona?

En muchos escenarios, es crucial asegurar dos cosas:

  1. Anonimato del participante: Garantizar que las respuestas no puedan ser rastreadas hasta un individuo específico.
  2. Unicidad de la respuesta: Prevenir que una misma persona responda múltiples veces.

¿Cómo funciona?

Code Sender actúa como un intermediario ciego que:

  • Genera códigos únicos y criptográficamente seguros
  • Envía cada código a un email específico
  • No almacena la relación entre el email y el código
  • Garantiza que el administrador no pueda saber qué código pertenece a qué persona

Características de Seguridad

  • API Key de SendGrid manejada de forma segura
  • Códigos generados con el módulo secrets de Python
  • Sin almacenamiento persistente de datos sensibles
  • Separación total entre identidad y código
